# Phase 2: Personal Reference

**Priority:** Medium
**Goal:** Polish the web view into a useful reference tool for KDM game sessions.
**Prerequisite:** Phase 1 (data is curated and editor is functional)

## Milestones

### 2.1 — Read-Only Browse Mode

- Polished card display (image + structured data in a clean layout)
- Fast search with instant filtering
- Keyboard navigation (arrow keys through items, `/` to focus search)

### 2.2 — Advanced Filtering

- Filter by multiple keywords (AND/OR)
- Filter by affinity color and position
- Filter by crafting location
- Filter by stat ranges (e.g., "strength >= 5")
- "What can I craft at [location] with [resources]?" query

### 2.3 — Gear Grid Helper

- Visual 3x3 grid where you can drag-and-drop gear
- Shows affinity connections between adjacent gear
- Highlights active affinity bonuses
- Save/load grid configurations

### 2.4 — Expansion Management

- Toggle which expansions are "in play" for the current campaign
- Filtered views only show gear from active expansions
- Persist expansion selection

## Done When

- Usable during a game session without friction
- Search and filter are fast enough to not break game flow
- Gear grid helper correctly calculates affinity connections
